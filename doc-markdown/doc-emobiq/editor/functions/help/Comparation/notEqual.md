# notEqual

	notEqual > (value1 = $value1, value2 = $value2)

## Description

Is a comparison operator that returns the boolean value true if both values are different and returns false otherwise.

## Parameters

**$value1** (Any)

The first value to be evaluated or checked.

**$value2** (Any)

The second value to be evaluated or checked.

## Return Value

**Boolean**
