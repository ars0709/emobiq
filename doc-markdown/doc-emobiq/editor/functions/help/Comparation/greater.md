# greater

	greater > (value1 = $value1, value2 = $value2)

## Description

Is a comparison operator that returns the boolean value true if the first value is greater than the second value and returns false otherwise.

## Parameters

**$value1** (Any)

The first value to be evaluated or checked.

**$value2** (Any)

The second value to be evaluated or checked.

## Return Value

**Boolean**
