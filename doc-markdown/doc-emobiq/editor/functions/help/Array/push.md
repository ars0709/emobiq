# push

	push > (var = $var_name, value = $value)

## Description

Insert new value inside an array.

## Parameters

**$var_name** (String)

The name of the variable that contains the main array list, this is where the new value will be inserted.

**$value** (Any)

The value to be added in the array.

## Return Value

**Array**
