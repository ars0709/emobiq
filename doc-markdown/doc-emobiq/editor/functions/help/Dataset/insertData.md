# insertData

	insertData > (	dataset = ($string),
	param = $object,
	callBack =success
	)

## Description

insert Data to emobiq staging database 

## Parameters

**dataset** = name of dataset (set on global services component)

**param** = use object type 
			<name field>=<value>,
			<name field>=<value>
**callBack** = event when success insert 


		
## Return Value

**boolean**
