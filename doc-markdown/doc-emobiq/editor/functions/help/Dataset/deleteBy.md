# deleteBy

	deleteBy > (	dataset = ($string),
	by = $string,
	value = $string
	)

## Description

delete record from local storage 
## Parameters

**dataset** = name of dataset (set on global services component)

**by** = name of field want to filter
**value** = content of field

		
## Return Value

**deleted**
