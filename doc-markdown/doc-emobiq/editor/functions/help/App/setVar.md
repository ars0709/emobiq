# setVar

	setVar > (var = $variable_name, value = $variable_value)

## Description

Create / Set a global variable along with its value.

## Parameters

**$variable_name** (Any)

The name to be assigned to the variable.

**$variable_value** (Any)

The value to be assigned to the variable.

## Return Value

**Any**
