# dbDate

	dbDate > (withTime = $time, add = $days)

## Description

Get the current date details.

## Parameters

**$time** (Boolean)

Indicates if the time will be included or not.

**$days** (Integer)

Additonal days to be included in the returned date details.

## Return Value

**Date/Time**
