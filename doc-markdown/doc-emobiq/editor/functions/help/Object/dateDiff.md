# dateDiff

	formatNumber > 
	(value1 = $date1,
	value2 = $date2,
	interval = $interval)

## Description

Get the difference of two dates.

## Parameters

**$date1** (Date/Time)

The data/time value to be substracted from the other value also called subtrahend.

**$date2** (Date/Time)

The date/time value will be the main value also called minuend.

**$interval** (Boolean)
