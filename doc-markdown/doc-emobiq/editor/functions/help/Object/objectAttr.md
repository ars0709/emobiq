# objectAttr

	objectAttr > (object = $object, attr = $attribute)

## Description

Get the attribute value of an object.

## Parameters

**$object** (Object)

The object where the attribute value will ba taken from.

**$attribute** (String)

The name of the attribute from the object.

## Return Value

**Any**
